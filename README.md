# vuex_todo_manager
this is a todo application implemented using vue js and vuex for state management with the help of JSON placeholder the fake rest api ( Todo Api )

functionalities: 

-Add new Todo

-Delete Todo

-Filter Todos

-Toggle the todo state between 'completed' and 'incompleted'

```
npm install
```
